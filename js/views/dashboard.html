Example action=parse

This page is using the parse action (action=parse) via the Wikipedia API to parse information from the Jimi Hendrix Wikipedia page. Information retrieved via the API, in JSON format, include the title, pageid and (some) of the content. In our case we only return the content of a specfic section by leveraging the section=$ parameter from the wikipedia API. By changing the API parameters you can influence which section are returned as part fo the parse action.

The content is returned, in our case, as HTML code, Wikipedia uses relative URLs instead of absolute URLs which means that all links are now pointing to the server where the Javascript is being served from and it is not pointing to the wikipedia.com website. When using this is a real-life situation you do want to strip the links from the content or ensure they are pointing to the correct (wikipedia.com) domain to ensure they are working correctly.
